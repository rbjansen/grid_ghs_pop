# grid_ghs_pop

> An aggregation of the [Global Human Settlement (GHS) framework's POP dataset](https://ghsl.jrc.ec.europa.eu/ghs_pop2019.php) dataset, to the [PRIO GRID](https://grid.prio.org/#/) dataset.

## Data
- grid_ghs_pop.geojson: GHS's global raster dataset of population, aggregated to the PRIO GRID resolution (~55x55km) by gid.
